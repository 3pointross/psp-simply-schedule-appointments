Give your clients, customers, and team the ability to schedule appointments right within the Panorama interface

== Description ==

Simply Schedule Appointments is an easy way to manage appointment scheduling right within WordPress. This integration will allow you to add a schedule an appointment button to Project Panorama, allowing your clients, customers, and team to book an appointment with you easy!

== Installation ==

1. Upload 'psp-simply-schedule-appointments' to the '/wp-content/plugins/' directory
2. Install Simply Schedule Appointments (free or premium)
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Setup your appointment types through Simply Schedule Appointments
5. Go to Projects > Settings > Add-ons > Simply Schedule Appointments and turn on appointment scheduling
